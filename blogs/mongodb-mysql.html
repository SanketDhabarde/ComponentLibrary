basic intro to mngodb

it is noSql database which makes it much flexible and most famous database among developers as compared to sql databases. here are some useful commands which we use in MongoDb.

  1. mongod
  2. mongo
  3. help
  4. show dbs
  5. use
  6. insert
  7. find
  8. update
  9. remove

we can see these commands one by one

1. mongod

it is primary deamon process for the MongoDb. to use the mongodb it is very usefull to start the mongod shell. you just need to type mongod in cmd and it will start the deamon process.


2. mongo

it is use to open the mongo shell. you just need to type mongo in your normal cmd and it will open up the mongodb shell. but for that you need to first set up the path variable.


3. help

as the name suggest it will help us to remember in case we forgot somethingin mongodb.we don't use this often but it is helpful to know. youjust needto type help in mongo shell.


4. show dbs

it is used to show the databases present in our system. you just need to type show dbs in the mongo shell.


5. use

is is use to use the selected database from the databases. also if the database is not present it will create the database and use it. you need to type use databse_name in the mongo shell.


6. insert

to insert the data in the database we use collections in the mongodb, unlike sql databases in which we use tables. for example, suppose we want to create the collection of dog in our database so to writhe the syntax you just need to type

db.dogs.insert({name: "Rusty", breed: "Mutt"})

to insert the data we use the object format i.e {property:value,..}


7. find

if we want to find all the items in our collection you just need to type db.dogs.find()
but if you want to find the specific item you need to pass the object like db.dogs.find({name:"Rusty"})


8. update

if you want to update some specific entries the update command take the two argument object first one is to select the entry which you want to update the second one is the values you wanna set to it. ex.

db.dogs.update({name:"Rusty"},{$set: breed:"labrodoodle", isCute:true})

it will change the breed and also add the new property.


9. remove

to remove the entry you just need to select the entry and delete it. ex.
db.dogs.remove({breed:"mutt"})
it will remove all the entries which having the breed mutt
